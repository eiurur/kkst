# Required

- <a href="https://github.com/yarnpkg/yarn" target="_blank">yarn</a>

# Setup

    git clone https://github.com/eiurur/kkst.git <project_name>
    cd <project_name>
    yarn install

# Usage

### Run

    yarn start

### Development

    yarn dev

### Test

    yarn test
